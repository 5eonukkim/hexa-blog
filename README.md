# hexa-blog

## To begin with

```bash
npm install hexo-cli -g
git clone https://github.com/HeXA-UNIST/hexa-blog.git
cd hexa-blog
npm install
```

## To write a post

```bash
hexo new "Codegate 2015 systemshock"
vi source/_posts/Codegate-2015-systemshock.md
```

## To publish your post

```bash
make deploy
```
